## 🎯 使い方

### カテゴリーでフィルタリング

サイドバーのカテゴリーをクリックすると、そのカテゴリーの予定のみが表示されます:

- **すべて**: 全ての予定を表示
- **仕事**: 仕事の予定のみ表示
- **各自の予定**: 個人的な予定のみ表示
- **共通の予定**: 家族共通の予定のみ表示

### 予定の追加

1. 「予定を追加」ボタンをクリック
2. カテゴリーを選択（デフォルトは「仕事」）
3. タイトル、日付などを入力
4. 保存

### 更新方法

#### 方法1: GitHubのWebインターフェース

1. GitHubリポジトリにアクセス
2. 各ファイルをクリック → 鉛筆アイコン（Edit）
3. ローカルの最新ファイルをコピー&ペースト
4. 「Commit changes」をクリック

#### 方法2: ファイルを再アップロード

1. GitHubリポジトリにアクセス
2. 「Add file」→「Upload files」
3. 3つのファイルをドラッグ&ドロップ
4. 「Commit changes」をクリック

※数分後に自動的に反映されます
